# 第一章 Unix 系统编程概述


## 1.2  什么是系统编程

程序模型

程序运行在计算机的内部,通过屏幕(monitor)输出,输入键盘(keyboard)输入数据,
执行的过程中,和内存(memory)进行数据存储与交互

